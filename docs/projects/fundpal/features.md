# 功能特性

FundPal 的功能设计围绕一个核心理念：**让投资者做出更理性的决策**。

## 📊 基金筛选

传统的基金筛选工具只提供简单的排名和评级。FundPal 支持多维度自定义筛选：

### 筛选维度

- **收益指标**：近 1 年 / 3 年 / 5 年收益率、年化收益
- **风险指标**：最大回撤、波动率、夏普比率
- **基金经理**：任职年限、管理规模、历史业绩
- **持仓分析**：重仓行业、前十大持仓、持仓集中度
- **费率对比**：管理费、托管费、申购赎回费

### 自定义策略

```typescript
// 示例：创建一个价值型基金筛选策略
const strategy = {
  name: '稳健价值型',
  filters: {
    maxDrawdown: { max: -20 },      // 最大回撤不超过 20%
    sharpeRatio: { min: 1.0 },      // 夏普比率 > 1
    managerYears: { min: 3 },       // 基金经理任职 > 3 年
    annualReturn3y: { min: 10 },    // 3 年年化 > 10%
  },
  sort: 'sharpeRatio',
  order: 'desc',
}
```

## 📁 组合管理

投资不是单只基金的事，组合管理才是关键。

### 核心能力

- **多组合支持**：可以创建多个投资组合（比如「养老组合」「成长组合」）
- **实时估值**：交易日实时估算组合净值变化
- **收益归因**：清晰展示每只基金对组合收益的贡献
- **再平衡提醒**：当资产配比偏离目标时自动提醒

### 可视化看板

- 资产配置饼图
- 收益曲线图
- 回撤热力图
- 行业 / 风格暴露矩阵

## 🔄 定投策略 <Badge type="warning" text="开发中" />

不只是「每月买固定金额」那么简单：

- **普通定投**：固定金额、固定日期
- **均线定投**：根据指数均线偏离度动态调整金额
- **估值定投**：根据指数 PE/PB 分位点调整投入
- **目标收益定投**：达到目标收益率后自动止盈

## 🤖 AI 分析 <Badge type="warning" text="开发中" />

结合大模型能力，提供有深度的分析：

- **基金点评**：输入基金代码，生成一份多角度分析报告
- **持仓诊断**：分析当前持仓的风险点和优化建议
- **市场解读**：将复杂的市场信息转化为通俗易懂的分析

## 📡 市场情绪 <Badge type="tip" text="规划中" />

- 主要宽基指数的 PE/PB 百分位
- 市场情绪温度计（基于成交量、涨跌比等）
- 北向资金流向追踪
