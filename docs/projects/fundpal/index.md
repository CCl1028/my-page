## 🎯 项目简介

**FundPal** 是我个人开发的基金投资助手工具，旨在帮助普通投资者更好地管理基金投资。它不是一个荐基平台，而是一个**辅助决策工具**——帮你看清数据、管好组合、守住纪律。

## 💡 为什么做这个项目

作为一个有投资习惯的开发者，我发现市面上的基金工具要么功能臃肿、要么充斥广告，很少有纯粹为「理性投资」服务的工具。于是我决定自己造一个：

- 📊 **数据驱动**：用数据说话，不靠感觉
- 🧠 **AI 辅助**：结合大模型能力，提供智能分析
- 🎯 **纪律约束**：帮你建立并遵守投资纪律
- 🔒 **隐私优先**：数据本地优先，不上传敏感信息

## ✨ 核心功能

| 功能 | 描述 | 状态 |
|------|------|------|
| 基金筛选 | 多维度筛选基金，支持自定义策略 | ✅ 已完成 |
| 组合管理 | 可视化管理投资组合，实时收益追踪 | ✅ 已完成 |
| 定投策略 | 智能定投计划，支持多种定投模式 | 🚧 开发中 |
| AI 分析 | 基于大模型的基金分析报告 | 🚧 开发中 |
| 市场情绪 | 市场温度计，辅助判断买卖时机 | 📋 规划中 |
| 回测系统 | 历史数据回测，验证投资策略 | 📋 规划中 |

## 🛠 技术栈

- **前端**：Vue 3 + TypeScript + Vite
- **UI 组件**：自研组件 + TailwindCSS
- **数据可视化**：ECharts
- **后端**：Node.js + Express
- **数据库**：SQLite（本地） / PostgreSQL（云端）
- **AI 能力**：OpenAI API / 本地模型
