# 我的投资体系搭建笔记

## 背景

作为一个程序员，我从 2022 年开始认真学习投资。从一开始追涨杀跌亏钱，到后来慢慢建立自己的投资体系，花了大概一年时间。

这篇文章不是教你怎么赚钱，而是记录**我搭建投资体系的思路和过程**。每个人的体系不同，关键是找到适合自己的。

## 问题

刚开始投资时踩的坑：

1. **没有体系**：看到什么热买什么，涨了不知道为什么赚，跌了不知道为什么亏
2. **情绪驱动**：大涨追进去，大跌割出来，完美反向操作
3. **不懂仓位管理**：要么满仓干，要么空仓看
4. **从不复盘**：同样的错误反复犯

## 方案

花了半年时间阅读经典书籍、学习优秀投资者的方法论，逐步搭建自己的框架。

## 实现

### 1. 投资哲学：先想清楚赚什么钱

| 赚钱方式 | 说明 | 我的选择 |
| --- | --- | --- |
| 赚企业成长的钱 | 买好公司，长期持有 | ✅ 主力 |
| 赚市场波动的钱 | 低买高卖，做差价 | ⚠️ 少量参与 |
| 赚信息差的钱 | 内幕、消息面 | ❌ 不参与 |

我的核心理念：**买股票就是买公司，赚钱来源是企业的长期价值增长。**

### 2. 选股框架

我用一个简单的 checklist：

- [ ] **行业好不好**：行业空间大吗？增速如何？竞争格局清晰吗？
- [ ] **公司好不好**：有没有护城河？管理层靠谱吗？财务健康吗？
- [ ] **价格好不好**：当前估值合理吗？有没有安全边际？
- [ ] **我懂不懂**：这个行业和公司，我能理解它的商业逻辑吗？

四个问题，前三个答「是」、第四个答「是」，才进入备选池。

### 3. 仓位管理

```
总仓位规则：
- 单只个股不超过总仓位的 20%
- 持仓不超过 5~8 只
- 保留 10%~20% 现金应对机会

加仓规则：
- 首次建仓不超过目标仓位的 1/3
- 下跌 10% 加仓 1/3（如果逻辑没变）
- 再跌 10% 加仓剩余 1/3

止损规则：
- 如果买入逻辑被证伪，无条件卖出
- 亏损超过 20% 必须重新审视
```

### 4. 复盘纪律

每月做一次复盘，记录以下内容：

1. **本月操作**：买了什么、卖了什么、为什么
2. **对错分析**：哪些决策是对的、哪些是错的
3. **情绪记录**：有没有被情绪驱动的操作
4. **持仓审视**：每只持仓的逻辑还成立吗
5. **下月计划**：接下来准备做什么

### 5. 推荐书单

入门阶段对我帮助最大的几本书：

1. 《投资最重要的事》 — Howard Marks
2. 《穷查理宝典》 — Charlie Munger
3. 《股市真规则》 — Pat Dorsey
4. 《手把手教你读财报》 — 唐朝
5. 《价值》 — 张磊

## 收获

1. **体系比技巧重要**：有体系才能在波动中保持定力
2. **认知圈决定收益**：只在自己懂的领域投资
3. **情绪是最大的敌人**：规则就是用来约束情绪的
4. **复盘是进步的关键**：犯错不可怕，重复犯同样的错才可怕
5. **耐心是最稀缺的品质**：好的投资机会不需要天天找

::: tip
投资和编程有很多相似之处：都需要系统思维、都需要持续学习、都需要在不确定中做决策。程序员做投资有天然优势——逻辑思维和数据分析能力。
:::
